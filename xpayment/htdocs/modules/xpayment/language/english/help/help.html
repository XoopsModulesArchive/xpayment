Help: xPayment

Description

Xpayment is a payment solution for XOOPS and XOOPS Clones, it allows you to have customise payment gateways and use Modular plugins to ensure any module can form submit an invoice for payment and fire plugins accordiantly to the status of the invoice as per the status of it enumerators!

Install/uninstall

Please make sure that you download the module from chronolabs to get the latest version, and upload it to your XOOPS Root directory from all the content inside the zips htdocs folder.

Intall the module as per the system module and then adjust preferences to suit, remember to set your Invoice Broker, Collection Officer and other groups accordiantly in the groups, this will be created as per your installation..
